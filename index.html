Transit Watch

GitHub repo

Developed for the Canadian Open Data Experience (CODE 2014), Transit Watch is an Android application that uses Canadian Open Weather datasets and NextBus API to provide users with ETAs for buses, and short-term weather forecasts. Transit Watch also paired with the Pebble API to create an interface for users on the Pebble Smartwatch.

Ultraviolet

Website

Hiding in plain sight.

Steganography is the art of concealing a message within an image in a virtually unnoticeable fashion, only to be revealed by those who are aware of the message's existence.

Ultraviolet is an intuitively light-weight and simple to use Google Chrome app that brings the art and stealth of steganography to the masses, made during the 48 hour PennApps hackathon in February 2014.
